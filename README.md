# Work Showcase 🗂️

A modern **portfolio hub** linking a selection of rebuilt web projects — by **[Governor Studio](https://governor-studio.vercel.app)**.

Each project was modernized into an up-to-date stack (Vite · React 19 · Tailwind v4 · Framer Motion): e-commerce, an interactive app, dashboards and product components.

## Featured
- **La Maison Jungle** — plant e-commerce (cart, filters)
- **Shiny** — freelance-matching app (survey → results)
- **Time Tracking** — interactive hours dashboard
- **Lumen** — waitlist landing with live validation
- **Analytics** — performance snapshot widget
- **Online Presence Audit** — channel score component

## Stack
**Vite · React 19 · Tailwind CSS v4 · Framer Motion**

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```

> Note: to serve as a GitHub Pages user site, this repo would need to be named `HayasMoustapha.github.io`; otherwise deploy on Vercel/Netlify.
